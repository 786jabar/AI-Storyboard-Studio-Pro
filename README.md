# AI Storyboard Studio Pro

Transform your stories into stunning visual storyboards using AI.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/786jabar/AI-Storyboard-Studio-Pro/blob/main/AI_Storyboard_Studio_Pro.ipynb)

## Features
- SDXL image generation (100% local, no API)
- SmolLM2 for story analysis
- 10 art styles + LoRA support
- Multi-panel storyboards
- PDF export
- Voice narration

## How to Run
1. Click the badge above
2. Select A100 GPU runtime
3. Run all cells
4. Use the ngrok URL
